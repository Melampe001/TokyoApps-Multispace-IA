## Project Components

### Android App (`app/`)
The Kotlin-based Android application providing Tokyo-themed AI features.

### Web (`web/`)
React/Vite web application with an admin panel.

### MCP Server (`server-mcp/`)
Node.js server implementing the Model Context Protocol (MCP).
